# OpenDictate

A local-first transcription setup focused on quick capture and low-latency edits.

## How it works

- Input watcher writes timestamped snippets to a local SQLite database.
- Post-processor cleans punctuation and casing using a lightweight on-device model.
- Daily archive keeps both raw and polished output in `~/.opendictate/archive/`.

## Setup

```bash
bun install
bun run dev
```

The watcher starts listening on the default audio input. Press `Ctrl+Space` to toggle recording.

## Notes

I built this because every cloud transcription service adds 200-400ms of latency which kills the flow when you're dictating code comments. Running Whisper locally on an M3 gives ~50ms end-to-end which is basically instant.

The archive format is intentionally simple — one JSON file per day with timestamps. Makes it easy to grep through later.
