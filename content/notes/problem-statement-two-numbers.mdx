# Problem Statement

You are given two numeric arrays and a target sum. Return any pair of indices whose values add up to the target.

## Approach

1. Clarify whether duplicates are allowed.
2. Prefer O(n) over O(n^2).
3. Return deterministic ordering for stable tests.

## Solution

```typescript
function twoSum(nums: number[], target: number): [number, number] | null {
  const seen = new Map<number, number>();

  for (let i = 0; i < nums.length; i++) {
    const complement = target - nums[i];
    const j = seen.get(complement);

    if (j !== undefined) {
      return [j, i];
    }

    seen.set(nums[i], i);
  }

  return null;
}
```

## Complexity

- **Time**: O(n) — single pass with hash map lookup.
- **Space**: O(n) — worst case stores all elements.

## Follow-up

What if the array is sorted? Use two pointers from both ends — O(1) space.
