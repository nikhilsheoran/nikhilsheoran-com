# Tally Server Setup

Setup cloudflare tunnel + wrap zero trust on server and all clients.

## Steps

1. Go to **Network > Connectors**, add tunnel (cloudflared), and enter private IP under CIDR.
2. Create users and security policies and then grant access.
3. Install cloudflared on all client machines.

## Windows Performance Tweaks

Type `sysdm.cpl`, go to **Advanced**, and set performance options for better remote responsiveness.

- Enforce removal of remote desktop wallpaper.
- Limit maximum color depth to 16 bit.
- Set RemoteFX adaptive graphics image quality to **Medium**.

## RDP Wrapper

For Windows Home editions that don't support RDP natively:

Reference: [rdpwrap releases](https://github.com/sebaxakerhtc/rdpwrap/releases)

Make sure to disable Windows Update before installing to prevent it from breaking the wrapper on next restart.

## Gotchas

- Cloudflare tunnels have a 100MB body size limit on free tier. Don't try to transfer large files through it.
- The zero trust WARP client sometimes conflicts with other VPNs. If you're running Tailscale, you need to split routes carefully.
